Prophet Cards This challenge had me style a card-sized profile of our prophet, Dallin H. Oaks. During this activity, I learned the basics of CSS, including how to size and position bocks, text and images.
Apostle Spotlight This challenge had me insert and position images of the First Presidency and Quorum of the Twelve. During this challenge, I learned how to insert images into HTML and position them in a grid.
Favorite Devotionals This challenge had me make a table in HTML and style it in CSS. During this challenge, I learned how to make tables, add content and style them.
Valentine's Day Page This challenge had me style a page to personal preference without changing the HTML. I learned how to position HTML elements in CSS.
Country Flags CSS This challenge had me fill grids with colors to make the shape of the flags of five different countries. I learned how to make and fill grids.
CSS positioning Activity This challenge had me position colors in a given way using grids and blocks. I learned how to position itmes on a page.
Image Gallery This challenge had me reduce the size of images to make the page load faster. I learned how to use Photopea to reduce image sizes.
Signup Form Design This assignment had me make and style a form. I learned how to make forms in HTML and style them in CSS.
